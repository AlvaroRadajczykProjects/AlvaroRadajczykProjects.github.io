LENGUAJES

RNA básica C++ NVIDIA CUDA 12.0

Con las librerías desarrolladas para este programa es posible entrenar una red neuronal artificial básica sin necesidad de implementar las operaciones matemáticas necesarias

Puedes encontrar el repositorio público de GitHub en este enlace


Detalles de uso

Se puede definir el número de valores de entrada y de salida de la red, su número de capas ocultas, el número de neuronas en cada capa oculta y la función de activación que utilizan.

También es posible entrenar a la red con el algoritmo estocástico del descenso del gradiente directamente o con el optimizador Adam, y se pueden definir el número de épocas, el tamaño de la batería, entre otros.

Además, es posible guardar todos los pesos y biases de la red entrenada en un archivo, y posteriormente exportarlo a una red con la misma estructura.

Para cambiar los datos de entrada y de salida del programa, es necesario programarlos desde el entorno de desarrollo (Visual Studio 2022). También es necesario para compilar el programa. Una vez programado, es posible quedarse con el archivo ejecutable compilado


Requisitos

Ordenador con una tarjeta gráfica con una capacidad computacional <= 3.0

Visual Studio 2022 con SDK para Windows 10 y herramientas para desarrollo de aplicaciones windows

NVIDIA CUDA Toolkit 12.0
