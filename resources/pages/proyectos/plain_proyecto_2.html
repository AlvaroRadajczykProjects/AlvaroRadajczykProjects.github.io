Cifrador Móvil documentos de texto

Con esta aplicación diseñada para Android con Java, es posible crear archivos de texto cuyo contenido está encriptado y solamente puede ser mostrado desde esta aplicación introduciendo la contraseña correcta. También es posible abrir y modificar dichos archivos las veces deseadas

Puedes encontrar el repositorio público de GitHub en este enlace


Detalles de uso

Esta aplicación guarda y abre todos los archivos en la carpeta de descargas del dispositivo móvil, no en cualquier directorio, por lo que se debe acceder a dicho directorio para encontrar los archivos encriptados


Requisitos

Dispositivo móvil (físico o virtual) con cualquier SSOO Android 11/SDK >= 30

Android studio, al menos la última versión estable
