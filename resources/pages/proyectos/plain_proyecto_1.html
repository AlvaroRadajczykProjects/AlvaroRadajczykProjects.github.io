Cifrador PC documentos de texto

Con esta aplicación diseñada para pc con Java y JavaSwing, es posible crear archivos de texto cuyo contenido está encriptado y solamente puede ser mostrado desde esta aplicación introduciendo la contraseña correcta. También es posible abrir y modificar dichos archivos las veces deseadas

Puedes encontrar el repositorio público de GitHub en este enlace


Detalles de uso

Esta aplicación permite elegir al usuario dónde guardar los archivos, y también permite buscarlos y abrirlos donde se desee


Requisitos

Java JDK >= 20
